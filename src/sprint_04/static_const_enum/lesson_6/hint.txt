Как использовать перечисления для определения команд в чат-боте?
Перечисления (enum) в Java — это удобный способ определить набор именованных констант.
 В контексте чат-бота перечисления можно использовать для определения команд, 
которые пользователь может ввести, чтобы взаимодействовать с ботом.
Например, ты можешь создать перечисление ChatCommand следующим образом:
```java public enum ChatCommand { MAP, RECYCLABILITY, BONUS }
`` Здесь MAP, RECYCLABILITY и BONUS` — это именованные константы, представляющие команды, которые может ввести пользователь.
Затем в коде чат-бота ты можешь использовать это перечисление для обработки команд:
```java String commandValue = scanner.nextLine();

 ChatCommand command = ChatCommand.valueOf(commandValue);
  switch (command) {
     case MAP: // Действия для команды MAP break;
     case RECYCLABILITY: // Действия для команды RECYCLABILITY break;
     case BONUS: // Действия для команды BONUS break;
      default: System.out.println("Такой команды нет.");
      }

`` В этом примере программа считывает строку, введённую пользователем, и преобразует её в объект типа ChatCommand.
 Затем с помощью оператора switch` она выполняет соответствующие действия в зависимости от значения команды.

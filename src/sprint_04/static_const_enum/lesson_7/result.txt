Переменная, объявленная внутри класса с модификатором static, называется статической,
 потому что она привязана исключительно к самому классу и существует независимо от его экземпляров.

  Для всего, что в программе менять нельзя, есть модификатор final.

Статический метод может обращаться только к статическим переменным или методам,
 и переопределять статические методы нельзя.
 Но можно объявить статический метод с одинаковой сигнатурой в родительском классе и классе-наследнике.

Внутри статического метода нельзя использовать ключевые слова this и super.

Если при объявлении переменной добавить модификатор final,
 то после инициализации её значение станет окончательным — изменить его будет нельзя.

Переменная с модификатором final уровня класса обязательно должна быть проинициализирована.
 Переменную с модификатором final необязательно инициализировать сразу.
 Это можно сделать в любой момент после её объявления и до первого применения в коде.

Константа в Java — это статическое финальное поле.
Чтобы его создать, нужно применить модификаторы static и final.
Для имён констант в Java принято использовать стиль SCREAMING_SNAKE_CASE.

Слово enum используется для объявления перечисления — типа данных,
который нужен для хранения ограниченного множества значений.
Все методы класса Object — toString(), getClass(), hashCode() и другие — можно применять к enum.
